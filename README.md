# Phaneendra Peravarapu Portfolio

**Live Link : https://phanee.site/**

A modern, responsive personal portfolio website built to showcase my profile, projects, work experience, internships, skills, resume, and contact links.

## Features

- Responsive single-page portfolio design
- Animated canvas background
- Name decoder animation in the hero section
- Typing animation for developer roles
- Dark and light theme toggle
- Resume view option
- Project filtering and searchable project cards
- Case-study modal for detailed project information
- Skill section with technology logos
- Redesigned work experience and internship cards
- Contact section with profile links and message form

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Devicon for technology logos
- Local SVG assets for project artwork and branding
